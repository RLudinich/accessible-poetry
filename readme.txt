== Description ==

Enriches your WordPress site with better accessibility options such as nicely Skiplinks, Font-Size changer, Contrast changer, Custom Toolbar and many other accessibility options for the front-end and the back-end of your site.

 What's included?
 
 * Accessibility toolbar - provides accessibility options for your users with contrast control, font-size changer & mark & underline links, stop animations and change to readable font.
 * Skiplinks - beautiful Skiplinks that gives your users the ability to skip between the main areas via the keyboard.
 * Missing ALT's - find all your images that don't have ALT attribute and let you set it.
 * Add language attribute to your <html> tag.
 * Add underline to all links.
 * Add special effect to element in focus mode (red or blue).
 * Force ALT attribute to all images.
 * Change link title to aria-label.
 * replace low heading tags (h4, h5, h6) with the h3 heading tag.
 * Add tabindex to all titles.
 
== Installation ==

1. Download the link.
2. Upload the zip file via the Plugin upload.
3. Activate the plugin.
4. Go to "Accessible Poetry" for more details.

== Frequently Asked Questions ==

= I activated the plugin but I can't see the toolbar of the skiplinks =

In order to display your toolbar or your skiplinks at the front of your site you'll need to place the dedicated functions for that at the right place. you'll find the dedicated functions beside the setting of it on the options panel page.
